# Falling Grade
DTI102 group tasks

ให้หลบเกรดที่ตกลงมา

รายละเอียดของเกม: จะมีเกรดตกลงมาจากข้างบนลงล่าง ให้ผู้เล่นหลบเกรด F ที่ตกลงมา และรับเกรด A, Project, Lab, Homrwork ที่ตกลงมา (เกรด F จะลบคะแนนครั้งละ 10 คะแนน เมื่อผู้เล่นโดนเข้า และเกรด A จะบวกคะแนนครั้งละ 5 คะแนน เมื่อผู้เล่นโดนเข้า) ถ้าคะแนนของผู้เล่นมีคะแนนเท่ากับ 100 ผู้เล่นจะชนะด้านที่ 1 ไปด่านที่ 2 ไปด่านที่ 3 และด่านที่ 4 จะเป็นด่านสุดท้าย แต่ความยากจะมากขึ้นเรื่อย ๆ เพราะจะมีเกรด F ตกลงมามากขึ้นและเร็วขึ้น และเกรด A จะตกลงมาน้อยลงเรื่อย ๆ ถ้าคะแนนผู้เล่นเหลือ 0 คะแนนจะจบเกม
